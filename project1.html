Toaster Timer

Toaster Timer is a pomodoro style productivity appwhich offers the cleanest UI
and the ability to easily track your study and break times every day.
Toaster Timer is a great way for users to just ‘get started’ and maximize their daily productivity.

Technologies: Toaster Timer is written in Java (Android Studio) and Swift (XCode).
It uses background services and core data as well as an SQLite and MySQL backend to store user progress.
